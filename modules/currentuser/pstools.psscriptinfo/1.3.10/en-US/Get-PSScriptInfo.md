# Get-PSScriptInfo

## SYNOPSIS

## SYNTAX

```
Get-PSScriptInfo [-FilePath] <FileInfo> [<CommonParameters>]
```

## DESCRIPTION
Collect and parse psscriptinfo from file

## EXAMPLES

### EXAMPLE 1
```
Get-PSScriptInfo -FilePath C:\temp\file.ps1
```

## PARAMETERS
